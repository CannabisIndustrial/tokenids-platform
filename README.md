# TokenIDS
## Plataforma para la Tokenización de Activos basada en Identidad Digital Soberana

**TFM — Master en Ingeniería Blockchain & IA**
Autora: Johana Niño Abella

## Stack
- Blockchain: Polygon PoS · Solidity 0.8.24 · ERC-1056 · ERC-3643
- Identidad SSI: W3C DID Core 1.0 · VC Data Model 2.0 · ZKP Groth16
- Backend: NestJS 10 · TypeScript · PostgreSQL 16 · Redis 7
- Frontend: Next.js 14 · wagmi v2 · RainbowKit v2
- IA selectiva: LangGraph
